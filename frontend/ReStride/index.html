Photography Tips

Ensure optimal input accuracy for the image segmentation model.

✓

Good Lighting

Bright, diffused lighting helps trace soft-tissue gradients accurately.

✓

Neutral Background

Solid, high-contrast backgrounds prevent edge classification errors.

✓

Entire Limb Visible

Include the proximal joint (e.g. knee) to establish anatomical orientation.

✓

Avoid Shadow Casts

Heavy shadows could be misclassified as tissue indentations.

✓

Capture 4-5 Angles

Include Front, Back, Left, Right, and Distal end perspectives.
